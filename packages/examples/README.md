# 💡 Ora Component Integration Examples

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

> Minimal, pure-integration templates showing how to import, configure, and mount **Ora Components** inside clean HTML and TypeScript apps.

---

## ⚡ Examples Included

*   **`index.html`**: A lightweight HTML layout demonstrating stylesheet and core bundle imports.
*   **Vite Integration**: High-performance setups using the Vite development server.
*   **RxJS Binding**: Clean structures displaying data stream mapping to labels and fields.

---

## 🚀 Getting Started

To spin up the example playground locally, run the following command from the monorepo root:

```bash
npm run dev --filter=ora-examples
```

Or run the dev server directly from this folder:

```bash
npm run dev
```

The example server will boot up on [http://localhost:5173](http://localhost:5173).
