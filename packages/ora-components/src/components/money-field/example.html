MoneyField Component Examples

Basic MoneyField with Multiple Currencies

// JavaScript usage: import { MoneyFieldBuilder } from 'ora-components/money-field'; import { BehaviorSubject } from 'rxjs'; const value$ = new BehaviorSubject({ amount: 1000, currencyId: 'USD' }); const field = new MoneyFieldBuilder() .withValue(value$) .withLabel('Transaction Amount') .withPlaceholder('Enter amount') .withCurrencies(['USD', 'EUR', 'GBP', 'JPY']) .build(); document.getElementById('example1').appendChild(field);

Single Currency (Static Symbol)

const field = new MoneyFieldBuilder() .withValue(new BehaviorSubject({ amount: 2500, currencyId: 'EUR' })) .withLabel('Euro Amount') .withCurrencies(['EUR']) // Single currency shows static € symbol .build();

With Error Message

const error$ = new BehaviorSubject('Amount exceeds maximum limit'); const field = new MoneyFieldBuilder() .withValue(new BehaviorSubject({ amount: 10000, currencyId: 'USD' })) .withLabel('Payment Amount') .withError(error$) .withMaxValue(5000) .withCurrencies(['USD', 'EUR']) .build();

Inline Error Mode

const field = new MoneyFieldBuilder() .withValue(new BehaviorSubject({ amount: -100, currencyId: 'USD' })) .withLabel('Withdrawal Amount') .withError('Amount cannot be negative') .withMinValue(0) .asInlineError() // Shows error icon instead of text below .withCurrencies(['USD', 'GBP']) .build();

With Constraints (Min/Max/Step)

const field = new MoneyFieldBuilder() .withValue(new BehaviorSubject({ amount: 100, currencyId: 'USD' })) .withLabel('Donation Amount') .withPlaceholder('$10 - $1000') .withMinValue(10) .withMaxValue(1000) .withStep(10) // Only multiples of 10 .withCurrencies(['USD']) .build();
